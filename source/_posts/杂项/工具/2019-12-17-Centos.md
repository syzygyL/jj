# 简易安装 Git 
## 更新 centos 
> 1| yum -y updata
## 快速安装命令
> 1| yum install git -y
## 设置基础账号与邮箱
> 1| git config --global user.name "你的账号"
2| git config --global user.email "你的邮箱"
# 安装 node 
## 安装 gcc 编译环境
> 1| yum install gcc-c++
## 下载源码包
> 1| wget https://npm.taobao.org/mirrors/node/v11.0.0/node-v11.0.0.tar.gz
## 解压源码包
> 1| tar -zxf node-v11.0.0.tar.gz
2| cd node-v11.0.0
## 编译 makefile 
> 1| ./configure && make
## 开始进行安装
> 1| make install
## 验证是否安装成功
> 1| node -v
2| npm -v
